## Mac for Flutter 
#### 代码补全插件 
Visual Studio Code (简称 VS Code / VSC) 
#### 系统快捷键设置
code -> perferce -> 键盘快捷键

command + r

##VSCode 插件 必备
Path Autocomplete : 路径智能补全插件。
Path Intellisense ： 路径智能提示插件。
Chinese (Simplified) Language Pack for Visual Studio Code ： 适用于 VS Code 的中文（简体）语言包。
